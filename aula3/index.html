    Subida de Servidor Apache

    Atribuir um endereço IP fixo para a nossa máquina:

    • Abrir o ubuntu server, digitar o comando cd /etc/netplan/ e um ls em sequência

    • Verificar se o arquivo 50-cloud-init.yaml existe

    • Digitar sudo su e inserir as credenciais do usuario

    • Digitar o comando nano 50-cloud-init.yaml

    • Apagar o conteúdo do arquivo deixando apenas a primeira linha (network) e digitar:

     version: 2

    renderer: networkd

    ethernets:

       enp0s3:

          dhcp4: no

          dhcp6: no

          addresses: [172.16.100.2/24]

          gateway4: 172.16.100.1

               addresses: [172.16.100.1,8.8.8.8]

    • Salvar o arquivo

    • Dar reboot para salvar as configurações

    • Após o reboot

    • Digitar sudo su e inserir as credenciais do usuario

    • Digitar ipconfig e observar o endereço que foi inserido anteriormente

    • Digitar ping 8.8.8.8 e conferir se está pingando

    • Atualizar os pacotes utilizando comandos sudo su apt update apt upgrade

    • Instalar o apache com o comando apt install apache2

    • Habilitar o apache no firewall com o comando ufw allow 'apache'

    • Checar o status com o comando systemctl status apache2

    • Verificar se o endereço da aplicação é o mesmo que foi configurado anteriormente com o comando 

       hostname -I

    • Parar o serviço com o comando systemctl stop apache2

    • Configurar o host virtual no diretório padrão do apache com o comando mkdir /var/www/osbrabo

    • Ir até o diretório cd /var/www

    • Atribuir permissões de uso com o comando chown -R $USER /var/www/osbrabo

    • Atribuir permissões de root com o comando chmod -R 755 /var/www/osbrabo

    • Editar a página inicial da aplicação com o comando nano /var/www/osbrabo/index.html

    • Inserir a sua página html:

       <!DOCTYPE html>
 <html lang="en">
 <head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>Bem vindo ao seu domínio!</title>
 </head>
 <body>
 <h1>Sucesso! Seu domínio virtual host está em funcionamento<\/h1\>
 <h5>ra ra ratinho-o-o-o<\/h5\>
 </body>
 </html>

    • Salve e feche o arquivo.

    • Reiniciar o serviço com o comando systemctl restart apache2
    • Configurar onde a aplicação vai funcionar com o comando nano /etc/apache2/sites-available/osbrabo.conf

    • Inserir os comandos:

       <VirtualHost *:80>
 ServerAdmin admin@admin
 ServerName osbrabo
 ServerAlias www.osbrabo
 DocumentRoot /var/www/osbrabo
 ErrorLog ${APACHE_LOG_DIR}/error.log
 CustomLog${APACHE_LOG_DIR}/access.log combined
 </VirtualHost>

    • Salve o arquivo e feche o arquivo

    • Recarregar o serviço com o comando systemctl reload apache2

    • Desabilitar o default com o comando a2dissite 000-default.conf

    • Digitar o comando apache2ctl configtest

    • Recarregar o serviço com o comando systemctl reload apache2

    • Ir na sua máquina com o windows 7 e testar digitando o endereço: 172.16.100.2
